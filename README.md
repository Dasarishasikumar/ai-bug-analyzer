# 🐞 AI Bug Analyzer + Task Generator

An AI-powered tool that analyzes software bugs, identifies root causes, assigns priority, and automatically creates tasks.

---

## 🚀 Features

- 🧠 AI-based bug analysis
- 🐞 Bug classification (Frontend / Backend / API / DB / DevOps)
- ⚡ Priority detection (High / Medium / Low)
- 👨‍💻 Auto task assignment
- 📝 Task title & description generation
- 🔗 Bugasura integration (auto-create bugs)
- 📊 Clean UI using Streamlit

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Ollama (LLM - llama3)
- Bugasura API
- Requests
